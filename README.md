# 🛡️ AI-Based Phishing Detection System

An intelligent cybersecurity solution that detects phishing attacks in emails, SMS, and URLs using Machine Learning and Natural Language Processing.

## 📌 Overview
This project aims to identify and prevent phishing attacks by analyzing suspicious text and links. It combines multiple datasets and machine learning models to classify inputs as **Safe**, **Suspicious**, or **Phishing**.

## ✨ Features
- 🔍 Email & SMS Phishing Detection
- 🌐 Malicious URL Detection
- 🤖 AI Chatbot Assistant (Explains results)
- 📊 Confidence Score Display
- 📁 Batch File Scanning
- 🎨 Interactive UI using Streamlit

## 🧠 Technologies Used
- Python
- Machine Learning (Scikit-learn)
- NLP (TF-IDF Vectorization)
- Streamlit (Frontend UI)


## 📂 Datasets Used
- Phishing Emails Dataset - https://www.kaggle.com/datasets/subhajournal/phishingemails
- Malicious URLs Dataset - https://www.kaggle.com/datasets/sid321axn/malicious-urls-dataset
- SMS Spam Collection Dataset - https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

## ⚙️ How It Works
1. User inputs text or URL
2. System detects language and translates (if needed)
3. Feature extraction using TF-IDF
4. ML models classify the input
5. Result is displayed with confidence score
6. Output is translated back and spoken using voice
